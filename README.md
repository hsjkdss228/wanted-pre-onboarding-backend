# 원티드 프리온보딩 백엔드 인턴십 선발 과제

## 개요

- 기업 채용을 위한 웹 서비스 서버 애플리케이션

## 기술 스택

- Spring Boot 3.1.4
- Spring Boot DevTools
- Spring Web
- Spring Data JPA
- H2 Database
- MySQL Driver
- Validation
- Flyway Migration
- Lombok
